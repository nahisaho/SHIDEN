# はじめに

この記事では、**SHIDEN（師伝）** で使えるサンプルプロンプトを紹介します。

SHIDENは、GitHub Copilotと連携して教育コンテンツを生成するAgent Skillsパッケージです。175件の教育理論と2,469セクションの学習指導要領を内蔵し、エビデンスに基づいた教育コンテンツを生成します。

:::note info
**SHIDENのインストールがまだの方は**
```bash
npm install shiden
npx shiden init
```
詳細は [SHIDEN紹介記事](https://qiita.com/hisaho/items/d15829049467f3ccb048) をご覧ください。
:::

# プロンプトの4つのパターン

SHIDENへのプロンプトは、大きく4つのパターンに分類できます。

| パターン | 説明 | SHIDENの動作 |
|----------|------|-------------|
| **基本例** | 標準的なリクエスト | 直接スキル実行 |
| **詳細例** | 具体的な条件を含む | 条件に合わせて生成 |
| **曖昧な例** | 情報が不足している | 1問1答でコンテキスト収集 |
| **特殊例** | 特別な配慮が必要 | 配慮事項を反映して生成 |

:::note warn
**曖昧なプロンプトでも大丈夫！**

「明日の授業の指導案を作って」のような曖昧なリクエストでも、SHIDENが1問1答形式で必要な情報を収集します。最初から完璧なプロンプトを書く必要はありません。
:::

# 📚 授業計画作成（lesson-plan）

## 基本例

```
中学2年生の数学で「一次関数」の授業計画を作成してください。
45分授業で、グラフの読み取りを中心にしたいです。
```

**期待される出力:**
- 学習目標（Bloom's Taxonomy基準）
- 時間配分付きの授業展開
- 学習指導要領との関連
- 参照した教育理論

## 詳細例

```
高校1年生の物理基礎「運動の法則」の授業計画をお願いします。

- 50分授業
- 生徒数: 35名
- 目標: ニュートンの運動の第2法則を理解し、簡単な計算問題が解けるようになる
- 実験器具: 台車、おもり、ばねばかりが使用可能
- 前時までに第1法則は学習済み
```

## 曖昧な例（1問1答フローが発動）

```
明日の授業の指導案を作ってください。
```

このプロンプトを送ると、SHIDENは以下のように質問してきます：

```
📝 あなたのリクエストを以下のように理解しました：

【元のリクエスト】明日の授業の指導案を作ってください
【強化されたリクエスト】対象学年・教科・単元を指定した授業計画の作成

📌 [Q1/5] 対象の学年と教科を教えてください。
（例: 「中学2年生の数学」「小学4年生の理科」）
```

## 特別支援を含む例

```
小学3年生の国語「物語の読み取り」の授業計画を作成してください。
クラスに読み書き障害（ディスレクシア）の児童が1名います。
音声教材とルビ付きプリントを活用したいです。
```

# 📝 教材作成（materials）

## ワークシート

```
小学4年生の理科「電気の流れ」のワークシートを作成してください。
実験の手順と考察欄を含めてください。
```

## スライド教材

```
中学1年生の社会「鎌倉幕府の成立」のスライド教材を作成してください。
10枚程度で、画像の説明や挿入位置の指示も含めてください。
視覚的にわかりやすいものをお願いします。
```

## クイズ・小テスト

```
高校2年生の英語コミュニケーション
Unit 5の文法事項（関係代名詞）を確認する小テストを作成してください。
- 問題数: 10問
- 形式: 空欄補充と並べ替え
- 所要時間: 10分程度
```

## 曖昧な例

```
来週使うプリントを作ってください。
```

## 複合教材（単元パッケージ）

```
小学6年生の算数「比例と反比例」の単元パッケージを作成してください。
- 導入用スライド（5枚）
- 練習問題ワークシート
- 振り返りシート
を含めてください。
```

# ✅ 評価設計（assessment）

## ルーブリック

```
高校1年生の英語プレゼンテーション用の5段階ルーブリックを作成してください。
発音、内容、構成、アイコンタクト、時間管理を評価項目にしてください。
```

**期待される出力例:**

| 観点 | 5 | 4 | 3 | 2 | 1 |
|------|---|---|---|---|---|
| 発音 | ネイティブに近い明瞭な発音 | 聞き取りやすい発音 | 概ね聞き取れる | やや聞き取りにくい | 聞き取りが困難 |
| ... | ... | ... | ... | ... | ... |

## 単元テスト

```
中学3年生の数学「二次方程式」の単元テストを作成してください。
- 配点: 100点満点
- 所要時間: 50分
- 難易度: 基本7割、応用3割
- 解答・解説も含めてください
```

## 形成的評価チェックリスト

```
小学5年生の国語「説明文の読み取り」の単元で使える
形成的評価のチェックリストを作成してください。
毎時間の振り返りに使いたいです。
```

## 曖昧な例

```
テストを作ってください。
```

## パフォーマンス評価

```
中学2年生の技術・家庭科（技術分野）
「プログラミングによる計測・制御」のパフォーマンス課題と評価基準を作成してください。
Scratch を使用し、センサーを活用した作品を制作します。
```

# 👤 個別指導計画（individual）

## 学習困難への対応

```
小学3年生の児童で、算数の計算（特に繰り下がりのある引き算）につまずいています。
個別指導計画を作成してください。
週1回30分の取り出し指導を想定しています。
```

## 発達障害への配慮

```
中学1年生のADHD傾向のある生徒への
英語学習支援計画を作成してください。
集中力の持続が難しく、ノートを取るのが苦手です。
通常学級での支援方法を中心にお願いします。
```

**期待される理論引用:**
- **Zone of Proximal Development（最近接発達領域）**
- **Universal Design for Learning（UDL）**
- **Scaffolding（足場かけ）**

## ギフテッド対応

```
小学5年生で数学的才能が突出している児童がいます。
通常の算数の授業中に取り組める発展課題と
長期的な学習支援計画を作成してください。
```

## 曖昧な例

```
支援が必要な生徒への対応を教えてください。
```

## 不登校傾向への対応

```
高校2年生で不登校傾向のある生徒の
学習保障計画を作成してください。
オンラインでの学習参加は可能で、
週1回のスクールカウンセラー面談を行っています。
現在、国語と英語の単位取得が危うい状況です。
```

# 💬 フィードバック（feedback）

## レポートへのコメント

```
高校2年生の現代文のレポート「羅生門の下人の心理分析」に対する
フィードバックコメントを作成してください。

【生徒の記述】
下人は最初は盗みをすることをためらっていたが、
老婆の話を聞いて盗みをしても仕方ないと思うようになった。
これは下人が悪い人間になったということだと思う。

良い点と改善点の両方を含め、次の学習につながるコメントにしてください。
```

## テスト返却時の声かけ

```
中学1年生の数学のテストで平均点を大きく下回った生徒への
声かけ・フィードバックの仕方を教えてください。
生徒は普段真面目に取り組んでいますが、今回は計算ミスが多かったです。
Growth Mindset を意識した声かけにしてください。
```

**期待される理論引用:**
- **Growth Mindset（成長マインドセット）** - Carol Dweck
- **Attribution Theory（原因帰属理論）**

## 振り返り支援

```
小学4年生の総合的な学習の時間
「地域の環境問題」のプロジェクト学習の振り返りシートへの
コメント例を作成してください。
自己評価と相互評価を促すフィードバックにしてください。
```

## 曖昧な例

```
生徒へのコメントを考えてください。
```

## ポートフォリオ評価

```
中学3年生の美術のポートフォリオに対する
年間を通した成長を認めるフィードバックを作成してください。
技術面だけでなく、表現の変化や意欲面にも触れてください。
```

# 🤝 生活指導案（guidance）

## いじめ対応

```
中学2年生のクラスで、SNS上での悪口・仲間はずれが発覚しました。
被害生徒、加害生徒、クラス全体への指導案を作成してください。
保護者への連絡文案も含めてください。
```

## 進路指導

```
高校3年生の進路に悩んでいる生徒との面談計画を作成してください。
成績は中程度で、大学進学と専門学校で迷っています。
本人は「やりたいことがわからない」と言っています。
キャリア理論を活用した面談の進め方をお願いします。
```

**期待される理論引用:**
- **Super's Career Development Theory**
- **Holland's RIASEC Model**
- **Planned Happenstance Theory**

## 学級経営

```
小学5年生の新学期最初の学級開きの指導案を作成してください。
学級目標の設定とルール作りを
児童主体で行いたいと考えています。
60分の学級活動を想定しています。
```

## 曖昧な例

```
問題行動のある生徒への対応を教えてください。
```

## 保護者対応

```
小学3年生の児童の保護者から
「うちの子が友達にいじわるをされている」という相談がありました。
保護者面談の進め方と、事実確認の手順、
その後の対応計画を作成してください。
```

## 危機対応例

```
高校1年生の生徒が「死にたい」とほのめかしています。
初期対応と関係機関との連携について指導案を作成してください。
```

:::note alert
**⚠️ 重要: 危機的状況での注意事項**

自傷、いじめ、虐待等の危機的状況が疑われる場合、SHIDENは必ず以下の警告を出力します：

> ⚠️ **重要**: この状況については、スクールカウンセラー、管理職、または専門機関への相談を強くお勧めします。AIの提案はあくまで参考情報であり、専門家の判断に代わるものではありません。

**AIの提案を鵜呑みにせず、必ず専門家と連携してください。**
:::

# プロンプトのコツ

## 1. 具体的な情報を含める

| ❌ 曖昧 | ✅ 具体的 |
|---------|----------|
| 授業計画を作って | **中学2年生の数学**で**一次関数**の授業計画を作って |
| プリントを作って | **小学4年生の理科**「電気の流れ」の**ワークシート**を作って |
| テストを作って | **高校1年生の英語**プレゼン用の**5段階ルーブリック**を作って |

## 2. 制約条件を明示する

```
✅ 良い例:
高校1年生の物理基礎「運動の法則」の授業計画をお願いします。

- 50分授業
- 生徒数: 35名
- 実験器具: 台車、おもり、ばねばかりが使用可能
- 前時までに第1法則は学習済み
```

## 3. 曖昧でもOK！

情報が不足していても、SHIDENが1問1答で聞き出してくれます。

```
⭕ これでも大丈夫:
明日の授業の指導案を作ってください。

→ SHIDENが「対象学年は？」「教科は？」「単元は？」と順番に聞いてきます
```

## 4. 特別な配慮は明記する

```
✅ 良い例:
クラスに読み書き障害（ディスレクシア）の児童が1名います。
音声教材とルビ付きプリントを活用したいです。
```

# まとめ

SHIDENを使えば、**曖昧なリクエストでも1問1答で情報を収集**し、**教育理論と学習指導要領に基づいた**質の高い教育コンテンツを生成できます。

| スキル | 用途 |
|--------|------|
| 📚 授業計画作成 | 指導案、時間配分、学習目標設定 |
| 📝 教材作成 | ワークシート、スライド、クイズ |
| ✅ 評価設計 | ルーブリック、テスト、チェックリスト |
| 👤 個別指導計画 | 学習困難、発達障害、ギフテッド対応 |
| 💬 フィードバック | コメント、振り返り支援 |
| 🤝 生活指導案 | いじめ対応、進路指導、学級経営 |

ぜひ、この記事のサンプルプロンプトを参考に、SHIDENを試してみてください！

```bash
npm install shiden
npx shiden init
```
